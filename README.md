

## To run this project locally on your own machine:

1. Clone the repository:

    bash
    git clone https://github.com/Rumman-Sadik/node-js-project.git
    cd chefer-express
    

2. Install the dependencies:

    bash
    npm install
   

3. Start the server:

   bash
    node server.js
    

4. Open your browser at:

   
   http://localhost:3000
